## Features

### Admin Features:
- Add/Remove/Update Products
- Add Product to Category
- Place Order
- View Addresses
### Customer Features:
- View Product
- Search Product
- Add Product to Cart
- Order Generate
- Add/Update/Delete Address







## Tech Stack

- Java
- Spring
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Postman
- Swagger UI
